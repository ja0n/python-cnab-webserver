# python-cnab-webserver
HTTP API Server for python-cnab project

Nesse momento só tem um endpoint para processar pagamentos DOC/TED do Banco do Brasil

## Run
```
git clone <>
cd python-cnab-webserver
pip install -r requirements
. run.sh
```
